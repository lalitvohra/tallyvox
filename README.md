# Tallyvox

**Digital Counter App** — A modern Android counter app for counting anything. Tap to increment, track multiple counters, and manage your counts with ease.

## Features

- Multiple counters with custom names
- Tap to increment / decrement
- Reset individual counters or all at once
- Clean, modern interface
- Fast and lightweight

## Tech Stack

- Android (Kotlin)
- Modern UI
